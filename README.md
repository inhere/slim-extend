## Install

- use composer

edit `composer.json`

_require_ add

```
"inhere/slim-extend": "dev-master",
```

_repositories_ add 

```
"repositories": [
        {
            "type": "git",
            "url": "https://git.oschina.net/inhere/slim-extend"
        }
    ]
```

run: `composer update`

## Usage

- add the middleware into slim application
